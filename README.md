# Substracker Landing Page

A self-contained Next.js landing page for the Substracker product.

## Setup

```bash
cd substracker
npm install
npm run dev
```

## Stack

- Next.js
- Tailwind CSS
- Framer Motion

## Notes

- The CTA buttons link to a placeholder download page and email contact.
- The landing page is designed for a light theme and modern conversion-focused layout.
